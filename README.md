## 🔍 Troubleshooting Common Issues

If the software fails to open, check the following points.

### The Program Does Not Start
Verify that your graphics card drivers are current. If you see an error about missing libraries, restart your computer. A restart resets the system paths and often fixes file access problems.

### Performance Is Too Slow
High-resolution molecule files create heavy work for the computer. If the application slows down, close other background programs that use high memory. Ensure the program has priority access to your graphics card.

### Task Fails During Processing
Some molecular files contain errors or broken geometry. Open the file in a standard viewer to check for missing atoms or overlapping structures. Clean files produce the best results.
